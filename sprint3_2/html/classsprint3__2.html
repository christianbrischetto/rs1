Detailed Description

sprint3_2 class

This class is used to read in an image, turn it into a cv::Mat data type and then produce a data type of nav_msgs::msg::OccupancyGrid which is publish to the topic /imageMap. This map is used to compared the true map of the simulation environment to the map produced via the SLAM toolbox
